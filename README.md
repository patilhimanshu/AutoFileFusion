# AutoFileFusion
This is a basic python code which autosyncs(merges) the content of multiple existing files in a folder into a new text file, the changes and content also visible in the terminal, This code uses watchdog library to detect and manage file changes.

This code repo also contains a similar branch which would filter the combined code, remove extra incantations and empty spaces beyond required count

✅ filesystem watcher
✅ automatic recompilation
✅ recursive-event prevention
✅ file aggregation
✅ configurable blank-line filtering
✅ OOP-based processing
✅ output generation
✅ event-driven updates
