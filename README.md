# AutoFileFusion.Filter
This is a basic python code which autosyncs(merges) the content of multiple existing files in a folder into a new text file, the changes and content also visible in the terminal, This code uses watchdog library to detect and manage file changes.
                                                                                AND
This code repo also contains a similar branch which would filter the combined code, remove extra incantations and empty spaces beyond required count. This new filtered data is stored in a new txt file created automatically which is named as "Filtered.txt".

The main goal of this code is to make shortened crisp notes with proper organized incantations, Thank you
